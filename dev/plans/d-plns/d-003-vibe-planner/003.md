## What This Plan Does

Transforms dev-docs from a generic chat tool into a purpose-built planning assistant. The user describes what they want to build in plain English. DeepSeek reads the actual code on disk via RAG, understands what already exists, and generates a structured plan in the exact dev-docs template format. The plan appears live in a preview panel and can be exported to disk as a ready-to-execute plan folder.

The core problem being solved: the user knows what they want but not how to articulate it technically. This app bridges plain-English intent and structured, agent-executable plans.

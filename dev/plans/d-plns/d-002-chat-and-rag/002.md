## Definition of Done

This plan is complete when:
- [ ] Settings screen has a "Test Connection" button that shows a green/red status badge
- [ ] Chat screen renders user and assistant message bubbles with markdown support
- [ ] User can send a message and see the assistant response stream in real time
- [ ] Input is disabled while streaming; Shift+Enter inserts a newline
- [ ] Chat history persists when navigating away and back (Zustand chatStore)
- [ ] Repos screen has a folder picker, Index button, and a live progress log
- [ ] Indexing writes a ChromaDB collection to disk (survives app restarts)
- [ ] Chat messages automatically include RAG context from the indexed folder
- [ ] No console errors during a full end-to-end session

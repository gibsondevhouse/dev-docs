# Stage 2 — Chat UI
**Plan:** 002 — Chat and RAG

## Stage Goal

The Chat screen is a fully functional streaming chat interface: user and assistant message bubbles, markdown rendering with syntax highlighting, real-time token streaming, and session-persistent history via Zustand.

## Stage-Level Setup (before PH1)

Install in the project root:

```bash
npm install react-markdown rehype-highlight highlight.js
npm install -D @tailwindcss/typography
```

Add typography plugin to `tailwind.config.js`:

```js
plugins: [require('@tailwindcss/typography'), require('tailwindcss-animate')],
```

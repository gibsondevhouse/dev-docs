- [ ] Agent builds a plan collaboratively via chat, outputting structured JSON
- [ ] Chat screen shows a live preview panel with the plan's file tree

- [ ] User confirms export and the full plan folder structure is written to disk
- [ ] Exported plan follows the exact template from `dev-docs/plans/_templates/`
- [ ] master-plan.md and active-plan.md are updated after export
- [ ] No errors in the browser console or Python sidecar logs during normal use

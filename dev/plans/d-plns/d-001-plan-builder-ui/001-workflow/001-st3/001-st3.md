# Stage 3 — RAG + Repo Indexing
**Plan:** 001 — Plan Builder UI

> **For the agent:** Before starting, confirm Stage 2 is complete (OpenRouter connectivity test passes). When this stage is done, update the changelog.

---

## Stage Goal

The user can select a local folder from the Repos screen, trigger indexing, watch progress in real time, and have the indexed content available as context for subsequent chat sessions. The RAG context is automatically injected into the system prompt when the user chats.

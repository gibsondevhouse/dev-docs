# Stage 7 — Export
**Plan:** 001 — Plan Builder UI

> **For the agent:** Before starting, confirm Stage 6 is complete (plan preview panel shows the file tree). When this stage is done, update the changelog.

---

## Stage Goal

The user can export the built plan to their local filesystem. An Export button opens a confirmation modal showing the target directory and file tree. On confirm, the Python sidecar writes the full folder/file structure to disk following the plan template exactly. `master-plan.md` and `active-plan.md` are updated automatically.

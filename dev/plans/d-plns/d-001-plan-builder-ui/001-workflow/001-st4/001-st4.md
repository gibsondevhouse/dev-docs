# Stage 4 — Chat UI
**Plan:** 001 — Plan Builder UI

> **For the agent:** Before starting, confirm Stage 3 is complete (RAG indexing works, context injection is wired). When this stage is done, update the changelog.

---

## Stage Goal

The Chat screen has a fully functional streaming chat interface: a scrollable message history showing user and assistant bubbles, a textarea input with send-on-Enter, and real-time streaming token display. Session history is preserved while the app is open.

---

## Entry Conditions

- [ ] Stage 3 is complete: RAG indexing and context injection are working
- [ ] `POST /chat` streams responses correctly

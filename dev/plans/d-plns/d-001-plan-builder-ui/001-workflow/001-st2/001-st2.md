**Plan:** 001 — Plan Builder UI

> **For the agent:** This document defines Stage 2 of the plan. Before starting, confirm Stage 1 is fully complete (native window opens, sidecar health check passes, routing works). When this stage is done, update the changelog.

## Stage Goal

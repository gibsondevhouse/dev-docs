# Stage 5 — Plan Builder Agent

## Stage Goal

The agent in the chat is specialized as a plan builder: it knows the plan template structure, asks clarifying questions to define a plan, and outputs structured JSON describing the full plan folder structure. The React app reads this JSON and populates a `planStore` that drives the preview panel.

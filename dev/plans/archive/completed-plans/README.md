# Completed Plans

Plans are moved here when every stage, phase, and part has been executed and the definition of done in `nnn.md` is fully checked off.

## How to Archive a Completed Plan

1. Move the plan folder from its type directory (`d-plns/`, `p-plns/`, `r-plns/`) into this folder
2. Remove the plan from `active-plan.md`
3. Update the plan's status to `Complete` in `master-plan.md`
4. Add an entry to the "Recently Completed" table in `roadmap/dev-map.md`

## What's Here

Plans in this folder are read-only reference. Do not modify them. They serve as a record of what was built and why.

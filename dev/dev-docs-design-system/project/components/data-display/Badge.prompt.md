**Badge** — small label/count pill. For plan status, prefer **StatusBadge**.

```jsx
<Badge variant="secondary" mono>d-002</Badge>
<Badge variant="success" dot>Connected</Badge>
<Badge variant="muted">2 plans</Badge>
```

Variants: `default` (dark), `secondary`, `outline`, `muted`, `success`, `destructive`. `mono` for IDs/counts/paths, `dot` adds a leading status dot.
